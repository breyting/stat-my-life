# Stat My Life

Une application pour suivre et visualiser les statistiques de votre vie (ex : nombre de fois que vous prenez le tram).

## Stack
- Next.js (site web)
- React Native (app mobile)
- Golang (API backend)
- PostgreSQL (base de donn√©es)

## Lancer le projet

### Backend
cd backend
go run main.go

### Site Web
cd web
npm install
npm run dev

### App Mobile
cd mobile
npm install
npm start
